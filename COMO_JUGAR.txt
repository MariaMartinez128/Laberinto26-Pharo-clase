COMO JUGAR AL LABERINTO

----------------------------------------
OBJETIVO DEL JUEGO
----------------------------------------

El objetivo es recorrer el laberinto, sobrevivir a los enemigos,
recoger objetos y llegar a la última habitación.

----------------------------------------
ESTRUCTURA DEL LABERINTO
----------------------------------------

El laberinto está formado por 6 habitaciones:

Hab-1 ─── Hab-2 ─── Hab-3
  │        │        │
Hab-4 ─── Hab-5 ─── Hab-6

----------------------------------------
ELEMENTOS DEL JUEGO
----------------------------------------

- Cofres → contienen objetos
- Altares → interacción especial
- Fosos → obstáculos
- Pociones → mejoran al jugador
- Bichos → enemigos

----------------------------------------
TIPOS DE ENEMIGOS
----------------------------------------

- Bicho Asustado → no ataca
- Bicho Frenético → ataca constantemente
- Otros posibles comportamientos dinámicos

----------------------------------------
COMANDOS DISPONIBLES
----------------------------------------

- norte → moverse hacia arriba
- sur → moverse hacia abajo
- este → moverse a la derecha
- oeste → moverse a la izquierda
- abrir → abrir puertas o cofres
- usar → utilizar objetos
- salir → terminar partida

----------------------------------------
SISTEMA DE TURNOS
----------------------------------------

El juego funciona por turnos:

1. El jugador realiza una acción
2. Los enemigos actúan
3. Se actualiza el estado

----------------------------------------
ESTRATEGIA
----------------------------------------

- Explorar todas las habitaciones
- Recoger pociones para mejorar habilidades
- Evitar enemigos peligrosos
- Usar objetos correctamente

----------------------------------------
FIN DEL JUEGO
----------------------------------------

El juego termina cuando:

- El jugador sale del juego
- El jugador pierde (si aplica)
- El jugador alcanza el objetivo final
